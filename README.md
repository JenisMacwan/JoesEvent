# JoesEvent

A modern event management website for creating excitement around events, sharing updates, and helping visitors request professional planning support.

Built with plain HTML, CSS, and JavaScript for easy GitHub Pages deployment.

## Sections

- Home hero with professional event imagery
- About section
- Services
- Featured events with category filters
- News and updates
- Contact request form

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md`.
3. Open the repository settings.
4. Go to **Pages**.
5. Set source to **Deploy from a branch**.
6. Choose the `main` branch and `/root` folder.
7. Save and wait for the public website URL.
